# Panorama Store

Panorama Store is a comprehensive e-commerce platform that offers a wide variety of products across numerous categories including electronics, clothing, furniture, kitchen appliances, and more. This project showcases a fully functional online store with key features that enhance user experience and streamline shopping activities.

## Features

- **Product Categories**: Includes mobiles, computers, watches, men's wear, women's wear, furniture, kitchen appliances, refrigerators, books, speakers, TVs, and ACs.
- **Cart Management**: Add items to the cart, remove items, and view cart contents.
- **Product Specifications**: Detailed product pages displaying price, brand, model, company name, product description, and version.
- **User-Friendly Interface**: Intuitive navigation and responsive design for seamless shopping experiences.
